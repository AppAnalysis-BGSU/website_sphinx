#  Building the project 
* Install sphinx 

`pip install Sphinx`

* Clone the repo 

`git clone https://github.com/AppAnalysis-BGSU/website_sphinx.git`

* Go to the root of the project and create html pages

`make html`

Acknowledgement: Standford Theme Package, 
pip install sphinx_theme
